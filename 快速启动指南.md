# 三件好事 - 快速启动指南

## 🚀 最简单的启动方式

### Windows 用户

双击运行：`启动项目.bat`

### 手动启动

#### 1. 启动后端（新终端窗口）

```bash
cd backend
npm install  # 首次运行需要
npm run dev
```

后端将运行在：http://localhost:3000

#### 2. 启动前端（新终端窗口）

```bash
npm run dev:h5
```

前端将运行在：http://localhost:5173

---

## ⚠️ 重要提示

### MongoDB 数据库

后端需要 MongoDB 数据库。有两种选择：

#### 选项 1：本地 MongoDB（推荐用于开发）

1. 下载安装：https://www.mongodb.com/try/download/community
2. 启动 MongoDB 服务
3. 后端会自动连接到 `mongodb://localhost:27017/three-good-things`

#### 选项 2：MongoDB Atlas（免费云数据库）

1. 注册：https://www.mongodb.com/cloud/atlas
2. 创建免费集群
3. 获取连接字符串
4. 修改 `backend/.env` 中的 `MONGODB_URI`

```env
MONGODB_URI=mongodb+srv://username:password@cluster.mongodb.net/three-good-things
```

## 🎯 功能说明

### 当前可用功能

✅ **游客模式**（默认）
- 无需登录，直接使用
- 数据保存在浏览器本地

✅ **邮箱注册/登录**
- 点击登录按钮
- 输入邮箱和密码
- 数据同步到云端

✅ **记录三件好事**
- 输入内容
- 自动保存
- 查看历史记录

✅ **多语言切换**
- 点击右上角语言切换器
- 支持中文/英文

✅ **响应式设计**
- 自动适配手机/平板/电脑

### 可选功能（需要配置）

⚪ **Google 登录**
- 需要配置 Google OAuth
- 编辑 `backend/.env` 填入 `GOOGLE_CLIENT_ID` 和 `GOOGLE_CLIENT_SECRET`

⚪ **AI 对话功能**
- 需要 OpenAI API Key
- 编辑 `backend/.env` 填入 `OPENAI_API_KEY`
- 点击右下角 LOGO 图标触发

## 🔧 常见问题

### Q1: 后端启动失败，提示 MongoDB 连接错误

**解决方案**：
```bash
# 检查 MongoDB 是否运行
# Windows: 打开服务管理器，查找 MongoDB 服务
# 或者使用 MongoDB Atlas 云数据库
```

### Q2: 前端无法连接后端

**解决方案**：
- 确认后端已启动（http://localhost:3000/health 应该返回 JSON）
- 检查 `.env` 文件中的 `VITE_API_URL` 配置

### Q3: 端口被占用

**解决方案**：
```bash
# 修改后端端口：编辑 backend/.env 中的 PORT
# 修改前端端口：编辑 vite.config.ts 中的 server.port
```
