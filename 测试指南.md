## 🔧 可能遇到的问题

### 问题1：找不到 app.json
**原因**：导入了错误的目录（项目根目录或 src 目录）
**解决**：必须导入 `dist/dev/mp-weixin` 目录

### 问题2：编译失败
**解决**：
```bash
# 停止当前编译
Ctrl+C

# 清理并重新安装依赖
rm -rf node_modules
npm install

# 重新编译
npm run dev:mp-weixin
```

### 问题3：页面空白或报错
**原因**：可能是缓存问题
**解决**：
1. 在微信开发者工具中点击"清缓存" → "清除全部缓存"
2. 重新编译项目

### 问题4：网络请求失败
**说明**：这是正常的！项目已配置离线模式
- 记录的好事会自动保存到本地
- 显示提示"已保存，稍后上传"
- 不影响功能测试

## 🎯 下一步开发建议

### 1. 后端 API 开发
需要实现以下接口：
- `POST /api/user/wx-login` - 微信登录
- `GET /api/user/info` - 获取用户信息
- `POST /api/record/submit` - 提交好事记录
- `GET /api/record/today` - 获取今日记录
- `GET /api/tree/info` - 获取树木信息

### 2. 配置真实 API 地址
修改 `.env.development` 文件：
```env
VITE_API_BASE_URL=http://你的后端地址:端口
```

### 3. 配置微信小程序 AppID
修改 `project.config.json`：
```json
{
  "appid": "你的真实AppID"
}
```

### 4. 添加 tabBar 图标（可选）
如果需要图标，在 `src/pages.json` 中添加：
```json
{
  "pagePath": "pages/index/index",
  "text": "首页",
  "iconPath": "static/tabbar/home.png",
  "selectedIconPath": "static/tabbar/home-active.png"
}
```
