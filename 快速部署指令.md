# 🚀 快速部署指令

## 一键命令

```bash
# 1. 重新编译项目
npm run dev:mp-weixin

# 2. 如果需要重新安装依赖
npm install
```

---

## 📝 必填配置项

### src/api/cloud.ts（第3行）
```typescript
const ENV_ID = 'cloud1-xxx' // ⚠️ 替换为你的云环境ID
```

## 🔄 文件替换清单

执行以下重命名操作（覆盖原文件）：

```bash
# Windows PowerShell
Move-Item -Path "src\store\user-cloud.ts" -Destination "src\store\user.ts" -Force
Move-Item -Path "src\store\record-cloud.ts" -Destination "src\store\record.ts" -Force
Move-Item -Path "src\store\tree-cloud.ts" -Destination "src\store\tree.ts" -Force

# 或者在 Git Bash / WSL
mv src/store/user-cloud.ts src/store/user.ts
mv src/store/record-cloud.ts src/store/record.ts
mv src/store/tree-cloud.ts src/store/tree.ts
```

## 📊 数据库集合清单

| 集合名称 | 权限设置 | 索引字段 |
|---------|---------|---------|
| users | 仅创建者可读写 | openid (唯一) |
| records | 仅创建者可读写 | userId, date, category |
| achievements | 仅创建者可读写 | userId, type |
| resonance | 所有用户可读，仅创建者可写 | - |

## ✅ 测试检查点

1. ✓ 打开小程序，能看到授权登录弹窗
2. ✓ 点击授权后，能看到用户昵称和头像
3. ✓ 提交三件好事后，能成功保存
4. ✓ 云开发控制台 → 数据库 → users 集合中能看到用户数据
5. ✓ 云开发控制台 → 数据库 → records 集合中能看到记录数据
