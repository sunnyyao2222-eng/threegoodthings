# 微信开发者工具导入指南

## ⚠️ 重要：正确的导入路径

**错误的做法：**
❌ 导入项目根目录：`E:\cluade3\three-good-things`

**正确的做法：**
✅ 导入编译输出目录：`E:\cluade3\three-good-things\dist\dev\mp-weixin`

## 📱 详细步骤

### 1. 打开微信开发者工具

### 2. 点击"导入项目"

### 3. 填写项目信息

- **项目目录**：选择 `E:\cluade3\three-good-things\dist\dev\mp-weixin`
- **AppID**：使用测试号或填入您的小程序AppID（wxcf6f4c781d54b799）
- **项目名称**：三件好事

### 4. 点击"导入"

## 🔧 如果遇到问题

### 问题1：找不到 app.json
**原因**：导入的目录不对
**解决**：确保导入的是 `dist/dev/mp-weixin` 目录，不是项目根目录

### 问题2：tabBar 图标不显示
**原因**：图标文件缺失
**解决**：暂时注释掉 tabBar 配置，或使用默认图标

### 问题3：编译错误
**原因**：开发服务器未运行
**解决**：确保 `npm run dev:mp-weixin` 正在运行

## 📂 目录结构说明

```
three-good-things/
├── src/                    # 源代码（不要导入这个）
├── dist/
│   └── dev/
│       └── mp-weixin/     # ✅ 导入这个目录
│           ├── app.json   # 小程序配置文件
│           ├── app.js     # 小程序入口
│           └── pages/     # 页面文件
```

## 🚀 快速修复

如果您已经导入了错误的目录，请：

1. 关闭当前项目
2. 重新点击"导入项目"
3. 选择正确的目录：`E:\cluade3\three-good-things\dist\dev\mp-weixin`

## ✅ 验证导入成功

导入成功后，您应该能看到：
- 左侧文件树显示 pages、static 等目录
- 模拟器中显示登录页面
- 控制台没有 "找不到 app.json" 的错误
