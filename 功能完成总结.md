## 📝 使用指南

### 1. 立即开始使用

打开浏览器访问：**http://localhost:5174**

- 自动以游客身份登录
- 开始记录你的三件好事
- 数据保存在浏览器本地

### 2. 注册正式账号

- 点击右上角"登录"按钮
- 选择"Sign Up"
- 输入邮箱、密码、昵称
- 数据自动同步到云端

### 3. 使用 AI 对话

- 点击右下角 LOGO 图标
- 展开 AI 对话窗口
- 输入问题，获得建议

### 4. 切换语言

- 点击右上角语言切换器（🌐）
- 选择中文或 English
- 界面立即切换

## 🔧 可选配置

### MongoDB 数据库

**选项 1：本地安装**
```bash
# 下载：https://www.mongodb.com/try/download/community
# 安装后启动 MongoDB 服务
```

**选项 2：MongoDB Atlas（免费云数据库）**
1. 注册：https://www.mongodb.com/cloud/atlas
2. 创建免费集群
3. 获取连接字符串
4. 修改 `backend/.env` 中的 `MONGODB_URI`

### Google OAuth

1. 访问 Google Cloud Console
2. 创建 OAuth 2.0 凭据
3. 配置回调 URL
4. 填入 `backend/.env`：
   ```env
   GOOGLE_CLIENT_ID=your-client-id
   GOOGLE_CLIENT_SECRET=your-client-secret
   ```

### OpenAI API

1. 访问 OpenAI Platform
2. 创建 API Key
3. 填入 `backend/.env`：
   ```env
   OPENAI_API_KEY=sk-your-api-key
   ```
