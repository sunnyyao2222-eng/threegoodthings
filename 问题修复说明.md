## ⚠️ 当前限制

### 没有 MongoDB 的影响
- ❌ **邮箱注册/登录**：不可用（需要数据库）
- ❌ **云端同步**：不可用（需要数据库）
- ✅ **游客模式**：完全可用（本地存储）
- ✅ **所有UI功能**：完全可用

### 如何启用完整功能

**选项 1：安装本地 MongoDB**
```bash
# 下载安装：https://www.mongodb.com/try/download/community
# 安装后启动 MongoDB 服务
# 后端会自动连接
```

**选项 2：使用 MongoDB Atlas（推荐）**
1. 注册：https://www.mongodb.com/cloud/atlas
2. 创建免费集群（512MB，永久免费）
3. 获取连接字符串
4. 修改 `backend/.env`：
   ```env
   MONGODB_URI=mongodb+srv://username:password@cluster.mongodb.net/three-good-things
   ```
5. 重启后端：
   ```bash
   cd backend
   npm run dev
   ```
