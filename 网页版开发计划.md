# 网页版开发计划

## 📋 技术方案

### 现有优势
- ✅ uni-app 已支持 H5 编译
- ✅ vue-i18n 已安装（国际化）
- ✅ Vue 3 + TypeScript
- ✅ Pinia 状态管理
- ✅ 现有UI组件可复用

### 需要调整的部分
1. **响应式布局**
   - 移动端：320px - 768px
   - 平板：768px - 1024px
   - 桌面端：1024px+

2. **国际化支持**
   - 中文（简体）
   - 英文
   - 日文（可选）
   - 韩文（可选）

3. **去除小程序特定API**
   - uni.login → Web登录方案
   - uni.setStorageSync → localStorage
   - 云开发 → REST API

4. **增强Web体验**
   - PWA支持
   - 响应式导航
   - 键盘快捷键
   - 深色模式

## 📱 响应式设计方案

### 断点设置
```scss
$mobile: 768px;
$tablet: 1024px;
$desktop: 1440px;
```

### 布局策略
- **移动端**：单列布局，底部导航
- **平板**：双列布局，侧边导航
- **桌面端**：三列布局，固定侧边栏
