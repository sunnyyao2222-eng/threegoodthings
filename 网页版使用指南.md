## 🌍 使用国际化

### 在组件中使用

```vue
<template>
  <view>
    <!-- 使用 $t 函数 -->
    <text>{{ $t('home.title') }}</text>

    <!-- 带参数 -->
    <text>{{ $t('home.remaining', { count: 2 }) }}</text>
  </view>
</template>

<script setup lang="ts">
import { useI18n } from 'vue-i18n'

const { t } = useI18n()

// 在 JS 中使用
console.log(t('common.confirm'))
</script>
```

### 切换语言

```typescript
import { changeLanguage } from '@/i18n'

// 切换到英文
changeLanguage('en-US')

// 切换到中文
changeLanguage('zh-CN')
```

### 添加语言切换器

```vue
<template>
  <LanguageSwitcher />
</template>

<script setup>
import LanguageSwitcher from '@/components/LanguageSwitcher.vue'
</script>
```

## 📱 使用响应式样式

### 在组件中引入

```vue
<style lang="scss" scoped>
@import '@/styles/responsive.scss';

.container {
  @include container; // 响应式容器

  @include mobile {
    // 移动端样式
    padding: 32rpx;
  }

  @include desktop {
    // 桌面端样式
    padding: 64rpx;
  }
}

.card {
  @include card; // 卡片样式
}

.button {
  @include button; // 按钮样式
}

.title {
  @include responsive-font(32rpx, 48rpx); // 响应式字体
}
</style>
```

### 常用混合宏

- `@include mobile` - 移动端样式
- `@include tablet` - 平板样式
- `@include desktop` - 桌面端样式
- `@include container` - 响应式容器
- `@include card` - 卡片样式
- `@include button` - 按钮样式
- `@include flex-center` - Flex居中
- `@include grid($columns)` - 响应式网格

## 💾 使用Web存储

import { storage } from '@/utils/web-storage'

// 保存数据
storage.set('user', { name: 'John', age: 25 })

// 读取数据
const user = storage.get('user')

// 删除数据
storage.remove('user')

// 清空所有数据
storage.clear()

## 🎨 响应式设计示例

### 首页布局

```vue
<template>
  <view class="home">
    <view class="header">
      <text class="title">{{ $t('home.title') }}</text>
      <LanguageSwitcher />
    </view>

    <view class="content">
      <!-- 内容 -->
    </view>
  </view>
</template>

<style lang="scss" scoped>
@import '@/styles/responsive.scss';

.home {
  @include container;
  min-height: 100vh;
}

.header {
  @include flex-between;
  padding: 32rpx 0;

  @include desktop {
    padding: 48rpx 0;
  }
}

.title {
  @include responsive-font(48rpx, 72rpx);
  font-weight: bold;
  color: #333;
}

.content {
  @include grid(1, 32rpx);

  @include tablet {
    grid-template-columns: repeat(2, 1fr);
  }

  @include desktop {
    grid-template-columns: repeat(3, 1fr);
  }
}
</style>
```

## 📦 部署

### 构建生产版本

```bash
npm run build:h5
```

### 部署到静态服务器

构建完成后，将 `dist/build/h5/` 目录部署到：

- **Vercel**：直接连接 GitHub 仓库
- **Netlify**：拖拽 dist 目录
- **GitHub Pages**：推送到 gh-pages 分支
- **自己的服务器**：上传到 Nginx/Apache

### Nginx 配置示例

```nginx
server {
    listen 80;
    server_name your-domain.com;
    root /var/www/three-good-things;
    index index.html;

    location / {
        try_files $uri $uri/ /index.html;
    }

    # 启用 gzip 压缩
    gzip on;
    gzip_types text/plain text/css application/json application/javascript;
}
```
